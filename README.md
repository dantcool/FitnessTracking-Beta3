# FitnessTracking

FitnessTracking is a multi-page web app for planning and tracking day-to-day fitness progress.

## App Type

Fitness tracking (workouts, meals/macros, body metrics, and profile goals)

## Problem It Solves

Helps users track meals, workouts, and weight in one place, with a dashboard view for quick navigation.

## Page Structure

1. Dashboard ([index.html](index.html))
	- Quick overview and tracker navigation.
	- Calendar view for monthly planning.

2. Meal Tracking ([page-2.html](page-2.html))
	- Breakfast, lunch, and dinner entries.
	- Macro inputs for protein, carbs, fat, and calories.
	- Dynamic extras section for snacks and additional food items.

3. Workout Tracker ([page-3.html](page-3.html))
	- Warm-up, strength, cardio, and cooldown sections.
	- Fields for sets/reps/weight, duration, intensity, and notes.
	- Dynamic extras section for additional exercises.

4. Weight Tracker ([page-4.html](page-4.html))
	- BMI calculator (metric and imperial support).
	- Weight log table with add/remove entries.

5. Profile ([page-5.html](page-5.html))
	- Personal info and body metrics.
	- Goal and preference settings.
	- Local save/reset profile behavior.

## Key Interactive Features

- Dashboard calendar and tracker shortcuts.
- Meal macro logging with dynamic extra meal entries.
- Workout logging with dynamic extra exercise entries.
- BMI calculation and weight history logging.
- Profile settings saved in browser storage.
